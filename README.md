## Overview
This repository contains manual QA test cases written to validate core functionality for the Sauce Demo Application.  
The goal of this project is to demonstrate practical QA skills, including writing test cases, executing tests, and documenting results.

## Scope
- Testing is done using a modern desktop browser  
- Test environment is a public demo site  
- Valid demo credentials are available  
- Testing focuses on:
  - Login functionality  
  - Products page functionality  
  - Cart functionality  
  - Checkout functionality
  - Logout funtionality
